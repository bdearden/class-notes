These examples, illustrating the truth and false of conditionals, are from my daughter Penny:

  • If Pikachu is an electric type, he can use thunderbolt.

    • True \(\to\)True is True

  • If Pikachu is an electric type, he can use hydro pump.

    • True \(\to\)False is False

  • If Pikachu is a duck, he can use thunder shock.

    • False \(\to\)True is True

  • If Pikachu is a grass type, he can use petal storm.

    • False \(\to\)False is True
