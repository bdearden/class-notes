Let \(A\) and \(B\) be two non-empty sets. A function (or mapping) from the set \(A\) to the set \(B\) is an assignment of exactly one element from \(B\) each element of the set \(A\text{.}\)

  • We denote this by \(f:A\to B\)
  • If \(b\in B\) is the unique element assigned to \(a\in A\text{,}\) we write \(f(a)=b\)
  • We call the set \(A\) the domain of the function, and the set \(B\) the codomain of the function.
