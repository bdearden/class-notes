Let \(A\) be an \(m\times k\) matrix, and let \(B\) be a \(k\times n\) matrix. Then the Boolean product \(A\odot B = \begin{bmatrix}c_{ij} \end{bmatrix}\) is defined by:

\begin{equation*} c_{ij} = ( a_{i1}\land b_{1j} ) \lor ( a_{i2}\land b_{2j} ) \lor \dots \lor ( a_{ik}\land b_{kj} ) \end{equation*}
