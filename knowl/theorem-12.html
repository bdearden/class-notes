Theorem 3.3.14. The Euclidean Algorithm.

Let \(a=r_0\) and \(b=r_1\) be two positive integers where \(a \ge b\text{.}\) If we apply the division algorithm recursively so that

\begin{equation*} r_j = r_{j+1} q_{j+1} + r_{j+2} \text{ where } 0 \le r_{j+2} \lt r_{j+1} \end{equation*}

for all \(j = 0, 1, 2, \dots, n-2\text{,}\) and \(r_{n+1} = 0\text{,}\) then \(\gcd(a,b) = r_n\text{.}\)
