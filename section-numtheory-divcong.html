Section 3.1 Divisibility and Congruences

¶
Note 3.1.1.

Any time we say “number” in the context of divides, congruence, or number theory we mean integer.

Subsection 3.1.1 The Divides Relation

¶
Definition 3.1.2.

Let \(a \and b\) be two integers with \(a \not= 0\text{.}\) We say \(a\) divides \(b\) and write \(a \divides b\) if there exists an integer \(m\) such that \(b = am\text{.}\)

We say that \(a\) is a factor of \(b\text{,}\) and \(b\) is a multiple of \(a\text{.}\)

Example 3.1.3.

The following are examples of the divides relation:

  1. \(3 \divides 6\) since \(6 = 3\cdot 2\)
  2. \(4 \divides 100\) since \(100 = 4 \cdot 25\)

Here are some non-examples:

  1. \(4 \nmid 10\) since there is no integer \(m\) for which \(10 = 4m\text{.}\)
  2. \(6 \nmid 3 \text{.}\) Order matters.
Proposition 3.1.4. Properties of divisibility.

Let \(a, b, c \in \Z\) with \(a \not= 0\text{.}\) Then:

  1. If \(a \divides b\) and \(a \divides c\) then \(a \divides ( b + c ) \text{.}\)
  2. If \(a \divides b\) then \(a \divides bc\) for all \(c\in \Z\text{.}\)
  3. If \(a \divides b\) and \(b \divides c\) then \(a \divides c \text{.}\)
Corollary 3.1.5.

If \(a, b, c\in \Z\) with \(a\not= 0\) such that \(a\divides b\) and \(a \divides c\text{,}\) then \({ a\divides (bm+cn)}\) where \(m \and n\) are some integers.

Theorem 3.1.6. The Division Algorithm.

Let \(n\) be an integer and \(d\) be a positive integer. Then there exist unique integers \(q \and r\) such that

\begin{equation*} n = dq + r \hspace{1em} \text{with} \hspace{1em} 0\le r \lt d \end{equation*}
Example 3.1.7.

Use the Division Algorithm to write each division as \(n = dq +r\) with \(0 \le r \lt d\text{,}\) with the variables defined in the statement above.

  1. \(543 \div 7\)
  2. \(-42 \div 13\)

Subsection 3.1.2 The Congruence Relation

¶
Definition 3.1.8.

Let \(a \and b\) be two integers and \(m\) be a positive integer. We say that \(a\) is congruent to \(b\) modulo \(m\) if

\begin{equation*} m \divides (b-a) \end{equation*}

We write \(a \equiv b \pmod m\)

We call \(m\) the modulus.

Example 3.1.9.
  1. \(13 \equiv 6 \pmod 7 \)
  2. \(-8 \equiv 6 \pmod 7 \)
  3. \(7 \equiv 0 \pmod 7\)
Definition 3.1.10.

If \(a\) is an integer and \(m\) is a positive integer, we define the congruence class of \(a\) modulo \(m\) to be the set of all integers congruent to \(a\) modulo \(m\text{.}\)

\begin{equation*} \left[ a \right]_m = \left\{ b \mid b \equiv a\pmod{m} \right\} \end{equation*}
Example 3.1.11.

What congruence class is indicated by the set:

  1. \(\left\{\dots, -6, -1, 4, 9, 14, \dots\right\}\)
  2. \(\left\{\dots, -13, 20, 53, 86, \dots \right\}\)
Example 3.1.12.

Find all congruence classes modulo 5.

Proposition 3.1.13.

Let \(m\) be a positive integer. Then integers \(a \and b\) are congruent modulo \(m\) if and only if there exists and integer \(k\) such that \({a = b + km}\text{.}\)

Proposition 3.1.14.

Let \(m\) be a positive integer. If \({ a \equiv b \pmod m}\) and \({c \equiv d \pmod m }\) then

\begin{equation*} a + c \equiv b + d \pmod m \hspace{1em} \and \hspace{1em} ac \equiv bd \pmod m \end{equation*}
Note 3.1.15.

For modular arithmetic, we have addition and multiplication. Subtraction is done via additive inverses, but division is not a thing! We'll discuss multiplicative inverses in a future section.

Subsection 3.1.3 Exercises

1.

Show that if \(a, b, \and c\) are integers with \(a\not=0 \and c \not= 0\) such that \(ac \divides bc\) then \(a\divides b\text{.}\)

2.

Use the division algorithm to write the following divisions as \(n = dq + r\) with the variables defined in the theorem:

  1. 17 is divided by 9
  2. 1234 is divided by 23
  3. 0 is divided by 13
  4. 8 is divided by 1
3.

Determine whether the following integers are congruent to 3 modulo 7:

  1. 37
  2. 66
  3. -17
  4. -67
  5. 80
4.

List all integers between -100 and 100 that are congruent to -1 modulo 25

5.

Suppose that \(a \and b\) are integers, \(a \equiv 4 \pmod{13}\) and \(b \equiv 9\pmod{13}\text{.}\) Find the integer \(c\) with \(0 \le c \le 12\) such that:

  1. \(c \equiv 9a \pmod{13} \)
  2. \(c \equiv 11b \pmod{13} \)
  3. \(c \equiv a+b \pmod{13} \)
  4. \(c \equiv a^2 + b^2 \pmod{13} \)
  5. \(c \equiv a^2 - b^2 \pmod{13} \)
Solution
  1. 10
  2. 8
  3. 0
  4. 6
  5. 11
6.

Find counterexamples to the following statements:

  1. If \(ac \equiv bc \pmod m\) where \(a, b, c, m \in \Z\) with \(m \ge 2\text{,}\) then \(a \equiv b \pmod m\text{.}\)
  2. If \(a \equiv b \pmod m\) and \(c \equiv d \pmod m\) where \(a, b, c, m \in \Z\) with \(m \ge 2\text{,}\) then \(a^c \equiv b^d \pmod m\text{.}\)

The first part of this exercise should convince you: you cannot “divide” both sides by \(c\) in modular arithmetic!

7.

Prove that 6 divides \(7^n - 1\) for all \(n \in \N\text{.}\)

Hint

(Alternative idea to the below proof) In the inductive step add and subtract \(7^k\text{.}\) That is, you'll have \(7\cdot 7^{k} - 1 - 7^k + 7^k\text{.}\) Now algebra.

Solution
Proof.

Let \(P(n)\) be the statement “6 divides \(7^n - 1\text{.}\)” We will show \(P(n)\) is true for all \(n \in \N\text{.}\) First we establish the base case, \(P(0)\text{.}\) Since \(7^0 - 1 = 0\text{,}\) and \(0\) is a multiple of 6, \(P(0)\) is true. Now for the inductive case. Assume \(P(k)\) holds for an arbitrary \(k \in \N\text{.}\) That is, 6 divides \(7^k - 1\text{,}\) or in other words, \(7^k - 1 = 6j\) for some integer \(j\text{.}\) Now consider \(7^{k+1} - 1\text{:}\)

\begin{align*} 7^{k+1} - 1 ~ \amp = 7^{k+1} - 7 + 6 \amp \text{by cleverness:} -1 = -7 + 6\\ \amp = 7(7^k - 1) + 6 \amp \text{factor out a 7 from the first two terms}\\ \amp = 7(6j) + 6 \amp \text{by the inductive hypothesis}\\ \amp = 6(7j + 1) \amp \text{factor out a 6} \end{align*}

Therefore 6 divides \(7^{k+1} - 1\text{,}\) or in other words, \(P(k+1)\) is true. Therefore by the principle of mathematical induction, \(P(n)\) is true for all \(n \in \N\text{.}\)
